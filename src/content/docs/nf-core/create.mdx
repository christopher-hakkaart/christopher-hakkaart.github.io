### Create a custom pipeline

To create a custom pipeline:

<Steps>

1. Run `nf-core pipelines create`.
2. Select **Let's go!**.
3. Select **Custom**.
4. Enter your pipeline details:
    - **GitHub organisation:** Specify your GitHub organisation.
    - **Workflow name:** Specify your pipeline name.
    - **A short description of your pipeline:** Describe your workflow.
    - **Name of the main author / authors:** Specify main author/authors name(s).
5. Select **Next**.
6. Select **Continue**.
7. Select **Finish**.
8. Once the pipeline is created, select **Continue**.
9. Select **Finish without creating a repo**.
10. Select **Close**.

</Steps>

See Template tour for more information about the files in the pipeline template.

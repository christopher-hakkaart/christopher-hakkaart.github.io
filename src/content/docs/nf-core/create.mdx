### Create a pipeline

<Steps>

1. Run `nf-core pipelines create`.
2. Select **Let's go!**.
3. Select **Custom**.
4. Enter your pipeline details then select **Next**
    - **GitHub organisation:** <ORGANISATION>.
    - **Workflow name:** <PIPELINE>.
    - **A short description of your pipeline:** <DESCRIPTION>.
    - **Name of the main author / authors:** <YOUR NAME>.
5. Select **Continue**.
6. Select **Finish**.
7. Once complete, select **Continue**.
8. Select **Finish without creating a repo**.
9. Select **Close**.

</Steps>
